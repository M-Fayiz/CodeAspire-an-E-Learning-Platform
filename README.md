# CodeAspire

### An Advanced Full-Stack E-Learning Platform

CodeAspire is a full-stack e-learning platform designed to connect learners with mentors through interactive learning experiences. The platform supports course enrollment, progress tracking, real-time communication, and one-to-one mentorship sessions.

---

## Key Features

- Browse and enroll in courses
- One-to-one video sessions between mentors and learners (WebRTC)
- Role-based access control (Admin / Mentor / Learner)
- Real-time chat using Socket.IO
- Learner progress tracking
- Certificate generation upon course completion
- AI-powered chatbot to clarify learner doubts
- High-performance backend with Redis caching
